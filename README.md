# CPL
# Semi-supervised Cross-modality Person Re-identification based on Pseudo Label Learning
This is the official implementation of our paper: Semi-supervised Cross-modality Person Re-identification based on Pseudo Label Learning.

### Prepare the datasets
SYSU-MM01: The SYSU-MM01 dataset can be downloaded from: http://isee.sysu.edu.cn/project/RGBIRReID.htm<br>
RegDB: The RegDB Dataset can be downloaded from: http://dm.dongguk.edu/link.html<br>
- Run pre_process_sysu.py for processing SYSU-MM01 and run pre_process_regdb.py for processing RegDB.
### Training
- run train_mine.py<br>
`--dataset`: regbd or sysu.<br>
`--margin`: triplet loss margin.<br>
`--gpu`: which gpu to run.<br>
### Results
The results on the SYSU-MM01 and RegDB datasets with 20% labeling rate.<br>
